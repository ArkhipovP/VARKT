## Структура репозитория

- `Apollo-11.craft` - файл KSP ракеты
- `code/` - весь Python-код проекта
  - `math_utils.py` - вспомогательные математические функции
  - `constants.py` - все константы, расчёт трансферного окна и высоты для завершения гравитационного манёвра
  - `autopilot.py` - kRPC-автопилот с телеметрией
  - `simulation.py` - наша физическая симуляция
  - `plotting.py` - визуализация сохранённых данных
- `data/` - все собранные данные в .csv формате
- `plots/` - все графики в формате .png

## Подготовка к запуску

1. Установка [Kerbal Space Program]().
2. Установка необходимых [модификаций]() (в папке `GameData`).
3. Установка [Python](https://www.python.org/downloads/).
4. Установка библиотек: `pip install -r requirements.txt` (в терминале).

## Использование

1. Создание сохранения в KSP, импортирование файла ракеты из репозитория.
2. Открытие меню kRPC Server, добавление и запуск сервера.
3. Запуск файла `autopilot.py`, ожидание успешного проведения автопилотирования. Сохранённые данные в файле `data/ksp_flight_data.csv`.
4. Запуск файла `simulation.py`, ожидание проведения симуляции. Сохранённые данные в файле `data/python_simulation_data.csv`.
5. Запуск файла `graphs.py`, лицезрение готовых графиков.
