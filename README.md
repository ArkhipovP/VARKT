## Структура репозитория

- `Apollo-11.craft` - файл KSP ракеты
- `code/` - весь Python-код проекта
  - `utils.py` - вспомогательные математические и файловые функции
  - `physics.py` - все физические расчёты и константы
  - `autopilot.py` - kRPC-автопилот с телеметрией
  - `simulation.py` - наша физическая симуляция
  - `plotting.py` - визуализация сохранённых данных
- `data/` - все собранные данные в .csv формате
- `plots/` - все графики в формате .png

## Подготовка к запуску

1. Установка [Kerbal Space Program]().
2. Установка необходимых [модификаций]() (в папке `GameData`).
3. Установка [Python](https://www.python.org/downloads/).
4. Создание виртуального окружения Python (через IDE или `python -m venv .venv`)
5. Активация виртуального окружения Python через IDE, `source .venv/bin/activate` (Linux/macOS) или `.venv\Scripts\activate.bat` (Windows).
6. Установка библиотек: `pip install -r requirements.txt` (в терминале).

## Использование

1. Создание сохранения в KSP, импортирование файла ракеты из репозитория.
2. Открытие меню kRPC Server, добавление и запуск сервера.
3. Активация виртуального окружения Python.
4. Запуск файла `autopilot.py`, ожидание успешного проведения автопилотирования. Сохранённые данные в файле `data/ksp_orbit_data.csv`.
5. Запуск файла `simulation.py`, ожидание проведения симуляции. Сохранённые данные в файле `data/sim_orbit_data.csv`.
6. Запуск файла `plotting.py`, лицезрение готовых графиков.

## Дополнительно

Запуск файла `physics.py` покажет вычисленные через Python значения некоторых физических величин.
